# NPM Commands

**npm install**
Run this first after cloning the repo to install all the requiremd dependencies for the project.

**npm run serve**
This will start a development server to test your code locally. When done, open http://localhost:8080 in a web browser to see/debug your app. Hot reload is supported so you can change and save files while this is running and see the changes automatically in the web browser. Note there are some random things that may not hot reload (usually having to do with CSS layout, margins, padding, etc). If this occurs, simply end the process (CTRL + C) and re-run the command.

**npm run build**
Build your code for use in production. This will create a 'dist' folder, within which will be the files for your app. This command will also automatically minify and reduce the file sizes to give you the smallest possible files.
